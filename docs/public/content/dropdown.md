# Dropdown

Dropdowns are also supported. You can use the `details` tag to create a dropdown. Example:

```html
<details>
  <summary>Open dropdown</summary>
  <ul>
    <li>
      <a href="#">Item 1</a>
    </li>
    <li>
      <a href="#">Item 2</a>
    </li>
    <li>
      <a href="#">Item 3</a>
    </li>
  </ul>
</details>
```

Right now icons are not supported in dropdowns. But will be available in the future.
